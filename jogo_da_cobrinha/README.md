# Snake Game

Este repositório faz parte de um projeto para a Digital Innovation One (DIO), lecionado pela *Front-End Developer* Gabriela Pinheiro. O desafio consta a recriação do famoso jogo da cobrinha (*snake game*) e foi utilizado a tecnologia **JavaScript**, juntamente com **HTML** e **CSS** no software Visual Studio Code.
